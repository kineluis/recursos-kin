# Recursos Educativos — Kinesiología 

Herramientas interactivas de acceso abierto para la enseñanza y el aprendizaje
en ciencias de la salud, desarrolladas por el docente klgo.Luis Flores Abarca 
## 🌐 Sitio web

👉 [kineluis.github.io/recursos-kin](https://kineluis.github.io/recursos-kin/)

## 🛠️ Herramientas disponibles

| Herramienta | Área | Descripción |
|---|---|---|
| [Laboratorio de Gasto Cardíaco](https://kineluis.github.io/recursos-kin/laboratorio_gasto_cardiaco_v2.html) | Fisiología cardiovascular | Simulador interactivo de gasto cardíaco y sus determinantes fisiológicos |
| [Entrenador de Objetivos Terapéuticos](https://kineluis.github.io/recursos-kin/entrenador_de_objetivos_terapeuticos_v5.html) | Razonamiento clínico | Redacción y evaluación de objetivos SMART con retroalimentación por IA |
| [Calculadora de 1RM — Brzycki](https://kineluis.github.io/recursos-kin/Calculadora_de_1rm_brzycki_v6.html) | Prescripción del ejercicio | Estimación de fuerza máxima y dosificación de carga por zonas de entrenamiento |
| [Dashboard de Mecánica Respiratoria](https://kineluis.github.io/recursos-kin/dashboard_respiratorio.html) | Fisiología respiratoria | Simulador interactivo de mecánica pleuro-pulmonar con monitor multicanal en tiempo real (presiones alveolar, pleural y transpulmonar, dinámica diafragmática y curvas del ciclo respiratorio) |
| [Constructor de Mapa CIF](https://kineluis.github.io/recursos-kin/CIF%20Kine.html) | Razonamiento clínico | Construcción asistida por IA de mapas CIF (Clasificación Internacional del Funcionamiento, la Discapacidad y la Salud) a partir de un caso clínico, con vista estudiante y panel docente |

## ⚙️ Características

- Funcionan directamente en el navegador, sin instalación ni registro
- Código abierto y modificable
- Diseño pedagógico basado en evidencia
- Algunas herramientas integran inteligencia artificial (requieren API key propia)
